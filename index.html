Installation

iCalc uses pip, the python package manager. You have pip automatically if your python version is greater than 3.0. To check your python version open up your terminal and run $ python -v. If your python version is less than 3.0 then installation depends on your operating system. For *NIX systems (darwin systems) like OSX and Linux you can install pip with python's older package manager easy_install. You run $ sudo easy_install pip. If you're on Windows however, you will need to download the python file get-pip.py from the project website and then run python on it plus some other stuff. The last time I checked the website's HTTPS certificate was invalid and I couldn't access it. Have fun. Your fault for choosing Windows (P.S.- you can try to get the package from here, but I don't know if it works).

Now that you have pip, you'll need to install virtualenv. The problem with running python files on different systems is that python versions can differ and lead to errors and incompatibility. To deal with this virtualenv creates an entire new virtual environment with its own python version and pip packages. To get virtualenv, run $ sudo pip install virtualenv if you're darwin based, and omit sudo if you're on Windows.

Now that you've done that, you can $ git clone https://github.com/JonW27/iCalc.git. Go to the directory via $ cd iCalc and run $ virtualenv venv. Then run $ source venv/bin/activate. This will take you into the virtualenv, marked by (venv) in the left margin of the terminal.

The final installation step is to run $ pip install -r requirements.txt. This will build the project by downloading the pip package requirements in order for the project to work.

Now that the installation is done you can run $ icalc --help

v0.1 iCalc calc options

sin(arg) - gets the sine of the parameter in radians.
cos(arg) - gets the cosine of the parameter in radians.
tan(arg) - gets the tangent of the parameter in radians.
sec(arg) - gets the secant of the parameter, or reciprocal of the cosine, in radians.
csc(arg) - gets the cosecant of the parameter, or reciprocal of the sine, in radians.
cot(arg) - gets the cotangent of the parameter, or reciprocal of the tangent, in radians.
fact(arg) - gets the factorial of the parameter, e.g. fact(3) = 3 * 2 * 1 = 6
sum(arg) - gets the summation of the parameter, e.g. sum(3) = 3 + 2 + 1 = 6
sqr(arg) - gets the square of the parameter; wrapper of python ** operator. E.g. sqr(5) = 25
sqrt(arg) - gets the square root of the parameter; its a wrapper of python ** (1/2). E.g. sqrt(25) = 5
deg2rad(arg) - converts the parameter from degrees to radians. E.g. 30 degrees becomes ~ 0.52 radians
deg2radf(arg) - converts the parameter from degrees to radians in terms of pi. E.g. 30 degrees becomes pi/6 radians.
fib(arg) - gets the parameter of the fibonacci sequence. E.g. fib(3) = 1,2,3 = 3. NOTE: The fibonacci sequence starts at 1.
prime(arg) - returns True if the parameter is prime; returns False if the parameter is not prime. E.g. prime(7)->True, prime(12)->False
floor(arg) - rounds the parameter down. E.g. floor(6.7)->6
ceil(arg) - rounds the parameter up. E.g. floor(6.3)->7
rand() - returns a random decimal value. NOTE: It does not have to be an integer.
randInt(lowerBound, upperBound) - returns a random integer value between lowerBound and upperBound. NOTE: due to a bug, does not work. Will be patched in v0.1.5
abs(arg) - returns the absolute value, or the parameter's distance from 0. E.g. abs(0) = 0
perm(n,r) - returns the amount of possible orders n objects at r time can be when order matters.
comb(n,r) - returns the amount of possible orders n objects at r time can be when order does not matter.
mod(x,y) - Gets the modulo of x/y. Wrapper for the python operator %. E.g. mod(12,5)-> 2
